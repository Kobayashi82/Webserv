This is a simple example to showcase Javascript file handling, and give a quick fake demo of how to add image upload handling in your prototype app. This code could be expanded to actually do something real, but what you're seeing isn't really uploading anything.

The label surrounding the input will activate it. Add display:none; to the input to hide it, and then design the label.

Create a new selector callback function for different utility uses. The previous example will replace itself, whereas the next example will prepend a new element each time.
